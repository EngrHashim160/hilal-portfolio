## Built with

This portfolio is built on the [Magic Portfolio](https://github.com/once-ui-system/magic-portfolio) template by Once UI (Next.js + MDX).

## Getting started

```bash
npm install
npm run dev
```

The site runs at `http://localhost:3000`.

To edit your content, update `src/app/resources/content.js` (bio, experience, skills, social links) and `src/app/resources/config.js` (theme, routes, base URL). Add projects as `.mdx` files in `src/app/work/projects/` and blog posts in `src/app/blog/posts/`.
