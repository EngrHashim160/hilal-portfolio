## Overview

An AI travel assistant that researches trips, compares options, and produces complete travel plans — then remembers your preferences for next time.

## What it does

- **Deep research**: Uses Tavily to research destinations and compare hotel and trip options.
- **Planning**: Generates itineraries, checklists, and budgets for each trip.
- **Memory**: Saves useful plans and preferences into Notion, with memory consolidation over time.
- **Interaction**: Communicates through Telegram for a chat-style planning experience.
- **Ops**: Configured automated weekly Hostinger server snapshots via n8n.

## Tech Stack

OpenClaw, Notion, Tavily, Telegram, n8n, Hostinger.

## Outcome

A personal travel agent that combines research, planning, and persistent memory into one assistant — turning vague travel ideas into structured, budgeted, ready-to-use plans.
