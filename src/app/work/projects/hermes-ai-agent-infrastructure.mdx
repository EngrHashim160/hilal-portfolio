## Overview

Deployed and hardened a Hermes AI Agent environment on a Hostinger Ubuntu VPS, with secure remote access and persistent memory across sessions.

## What I Built

- **Secure Access**: Hermes WebUI placed behind Cloudflare Tunnel and email-based Cloudflare Access.
- **Stable Routing**: Stabilized Docker port mapping for reliable routing after server restarts.
- **Self-Hosted Memory**: Stood up a Honcho memory backend with PostgreSQL/pgvector and Redis.
- **Persistent Memory**: Connected Honcho to Hermes for persistent cross-session memory.
- **Documentation**: Authored operational runbooks and client handoff documentation.

## Tech Stack

Hostinger Ubuntu VPS, Docker, Cloudflare Tunnel, Cloudflare Access, Hermes, Honcho, PostgreSQL/pgvector, Redis.

## Outcome

A secure, restart-resilient, memory-enabled AI agent infrastructure with proper access control and complete operational documentation ready for client handoff.
