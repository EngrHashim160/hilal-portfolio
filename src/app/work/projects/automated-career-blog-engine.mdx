## Overview

An autonomous content engine that produces and publishes a steady stream of SEO-optimized articles every day with no manual writing.

## What it does

- **Content creation**: Generates 10 SEO-optimized Arabic articles daily using an LLM.
- **Imagery**: Creates photorealistic images to accompany each article.
- **Publishing**: Publishes finished posts to WordPress automatically via n8n.

## Tech Stack

n8n, LLM content generation, AI image generation, WordPress.

## Outcome

A fully automated daily publishing pipeline that scales content production far beyond what manual writing allows — consistent, SEO-friendly, and multilingual.
