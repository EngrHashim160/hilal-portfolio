## Overview

The Web Scrap Agent is an automated n8n workflow that keeps a WordPress site stocked with fresh, structured job content with minimal manual effort.

## How it works

- **Scrapes** job postings from multiple websites on a schedule.
- **Rewrites** each posting in Arabic using AI, with SEO-friendly formatting.
- **De-duplicates** against Airtable so the same job is never posted twice.
- **Publishes** finished posts directly to WordPress as drafts or live posts.

## Tech Stack

n8n, AI rewriting (LLM), Airtable, WordPress REST API, Google Gemini.

## Outcome

A hands-off content pipeline that delivers fresh, SEO-optimized, multilingual job listings on a schedule — turning scattered job sources into a single structured publishing flow.
