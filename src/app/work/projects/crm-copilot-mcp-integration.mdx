## Overview

A set of AI copilots that bring CRM operations into a conversational interface, letting users manage contacts, deals, pipelines, and tasks through natural language instead of manual UI work.

## GoHighLevel + Claude Desktop (MCP)

Integrated GoHighLevel (GHL) with Claude Desktop through the Model Context Protocol (MCP), enabling natural-language control of CRM operations directly from Claude. Provided real-time access to contacts, opportunities, pipelines, tasks, notes, and communication workflows.

## HubSpot + OpenClaw + WhatsApp

Built an AI CRM copilot integrating HubSpot CRM with OpenClaw and a WhatsApp channel, enabling fully conversational CRM management. Implemented workflows for contact management, deal updates, follow-ups, activity retrieval, and real-time interactions — all hands-free over WhatsApp.

## Tech Stack

GoHighLevel, HubSpot, Claude Desktop, Model Context Protocol (MCP), OpenClaw, WhatsApp.

## Outcome

Two production CRM copilots that demonstrate how MCP and agentic tooling can turn complex CRM platforms into natural-language, conversational experiences for daily business operations.
