## Overview

An end-to-end Streamlit application that demonstrates agentic AI patterns built on LangGraph and LangChain, designed for rapid experimentation with multiple agent modes.

## Modes

- **Basic Chatbot**: A straightforward conversational agent.
- **Web-Augmented Chatbot**: Uses Tavily tool calls to fetch live information.
- **AI News Summarizer**: Summarizes news and exports the result as markdown.

## Architecture

- Configurable via INI files
- Powered by Groq LLMs
- Modular nodes and tools for composable agent graphs
- Clean Streamlit UI for fast iteration

## Tech Stack

LangGraph, LangChain, Groq LLMs, Tavily, Streamlit.

## Outcome

A reusable template for building and experimenting with agentic workflows, cleanly separating the graph logic, tools, and UI so new agent behaviors can be added quickly.
