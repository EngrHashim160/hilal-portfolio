## Overview

A coordinated set of three OpenClaw agents that automatically analyze and optimize a YouTube channel.

## The agents

- **Insight Agent**: Analyzes the channel and points out its weaknesses.
- **SEO Agent**: Optimizes and updates the metadata of videos.
- **Thumbnail Agent**: Updates or creates new thumbnails for videos.

## The flow

The Insight Agent runs first and identifies what's holding the channel back. Based on its findings, it recommends the SEO Agent for metadata optimization and the Thumbnail Agent for thumbnail updates or new creation — so optimization is targeted, not blind.

## Tech Stack

OpenClaw, multi-agent orchestration, YouTube metadata and thumbnail tooling.

## Outcome

A self-directing optimization loop where analysis drives action — diagnosing channel weaknesses and then fixing metadata and thumbnails through specialized agents.
